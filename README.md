# GopherHound
A simple tool to convert any Go repository into a BloodHound OpenGraph compatible payload.

# Installation
```bash
go install github.com/wes-mil/GopherHound
```

# Usage
Navigate to the project root and run
```bash
GopherHound --output opengraph.json
```
